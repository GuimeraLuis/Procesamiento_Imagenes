---------------------------------------
De que se trata?
Se trata de un script con extensión .ipynb donde se utilizan distintos métodos y funciones de algunas librerías de Python para trabajar con imagenes  
---------------------------------------
Que contiene?

Lab 01 - Espacio de color.ipynb: contiene una serie de métodos y funciones para el análisis del espacio de color de una imagen, como por ejemplo la descomposición en canales, la identificación de bordes de objetos, el suavizado o desenfoque de una imagen
---------------------------------------
Como se usa?
1 - Se comienza cargando las librerías a utilizar
2 - Se carga una imagen a elección
3 - Se aplica los distintos métodos a la imagen cargada y se van visualizando sus avances
