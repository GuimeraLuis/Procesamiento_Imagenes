---------------------------------------
De que se trata?
Se trata de un script con extensión .ipynb donde se utilizan distintos métodos y funciones de algunas librerías de Python para 
aprovechar modelos pre-entrenados en grandes conjuntos de datos y clasificar el objeto en una imagen
---------------------------------------
Que contiene?

010 - Transferencia de Aprendizaje con ResNet18.ipynb: contiene una serie de métodos y funciones para:
- predecir a que clase pertenece el objeto de una imagen 


---------------------------------------
Como se usa?
1 - Se comienza cargando las librerías a utilizar
2 - Se carga un modelo pre-entredado
3 - Se carga la clases a utilizar en predicción
4 - Se carga imagen a procesar
5 - Se obtiene resultado
