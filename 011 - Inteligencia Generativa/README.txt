---------------------------------------
De que se trata?
Se trata de un script con extensión .ipynb donde se utilizan distintos métodos y funciones de algunas librerías de Python para 
aprovechar modelos pre-entrenados para la generación de imágenes
---------------------------------------
Que contiene?

011 - Generación de Imagen con Stable Difusion.ipynb: contiene una serie de métodos y funciones para:
- a partir de un prompt generar una imagen
- solicitar prompt y mostrar resultado a través de Gradio 


---------------------------------------
Como se usa?
1 - Se comienza cargando las librerías a utilizar
2 - Se carga un modelo pre-entredado
3 - Se genera interfaz con Gradio
4 - Se ingresa prompt y se espera resultado
5 - Se obtiene resultado
