De que se trata?
Se trata de un script con extensión .ipynb donde se utilizan distintos métodos y funciones de algunas librerías de Python para trabajar con imagenes  
---------------------------------------
Que contiene?

Lab 02 - Muestreo Espacial.ipynb: contiene una serie de métodos y funciones para poder aplicar el proceso de muestreo espacial de una imagen, que consiste en la extracción de un pixel cada cierta cantidad de pixeles que se indique (factor de muestreo) y luego utilizar los pixeles extraidos para la composición de una nueva imagen.
---------------------------------------
Como se usa?
1 - Se comienza cargando las librerías a utilizar
2 - Se carga una imagen a elección
3 - Se aplican distintos métodos y ciclos para la extracción de pixeles de la imagen seleccionada. 
4 - Se utilizan los pixeles extraidos para la creación de una nueva imagen
5 - Se visualizan las distintas imágenes creadas con distintos factores de muestreo para poder apreciar la técnica
