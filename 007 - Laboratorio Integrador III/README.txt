De que se trata?
Se trata de un script con extensión .ipynb donde se utilizan distintos métodos y funciones de algunas librerías de Python para trabajar con imagenes  
---------------------------------------
Que contiene?

Lab 07 - Ejercicio Integrador.ipynb: contiene una serie de métodos y funciones para:
- el reconocimiento de rostros en imagenes
- el reconocimiento de landmarks (puntos faciales) en rostros

---------------------------------------
Como se usa?
1 - Se comienza cargando las librerías a utilizar
2 - Se carga una imagen a elección
3 - Se carga un modelo haardcascade
4 - se utiliza el método para reconocer rostros y guardar coordenadas
5 - se dibuja rectángulo con coordenadas obtenidas
6 - Se visualizan las distintas coordenadas en la imagen original
7 - Se utiliza método para reconocimiento de landmarks en rostros detectados
8 - Se dibujan puntos obtenidos en imagen original
