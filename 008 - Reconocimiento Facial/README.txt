---------------------------------------
De que se trata?
Se trata de un script con extensión .ipynb donde se utilizan distintos métodos y funciones de algunas librerías de Python para identificar landmarks faciales imágenes y mostrar los mismos   
---------------------------------------
Que contiene?

Lab 08 - Reconocimiento Facial con MediaPipe.ipynb: contiene una serie de métodos y funciones para:
- el reconocimiento de rostros en imagenes
- el reconocimiento de landmarks en rostros para el dibujo de mallas faciales

---------------------------------------
Como se usa?
1 - Se comienza cargando las librerías a utilizar
2 - Se carga una imagen a elección
3 - Se carga un modelo facemesh
4 - Se detectan los puntos faciales
5 - se dibuja malla facial a partir landmarks obtenidos
