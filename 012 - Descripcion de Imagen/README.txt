---------------------------------------
De que se trata?
Se trata de un script con extensión .ipynb donde se utilizan distintos métodos y funciones de algunas librerías de Python para 
aprovechar modelos pre-entrenados para la descripción de imágenes
---------------------------------------
Que contiene?

012 - Descripcion de Imagen - Blip.ipynb: contiene una serie de métodos y funciones para:
- a partir de una imagen generar la descripción de la misma
- solicita cargar la imagen y muestra el resultado a través de Gradio 


---------------------------------------
Como se usa?
1 - Se comienza cargando las librerías a utilizar
2 - Se carga un modelo pre-entredado
3 - Se genera interfaz con Gradio
4 - Se carga imagen y se espera resultado
5 - Se obtiene resultado
