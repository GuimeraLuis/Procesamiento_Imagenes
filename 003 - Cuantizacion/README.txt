De que se trata?
Se trata de un script con extensión .ipynb donde se utilizan distintos métodos y funciones de algunas librerías de Python para trabajar con imagenes  
---------------------------------------
Que contiene?

Lab 03 - Cuantizacion.ipynb: contiene una serie de métodos y funciones para poder aplicar el proceso de cuantizacion de una imagen, que consiste en disminución de los niveles de grises aplicando un factor de cuantizacion para la composición de una nueva imagen.
---------------------------------------
Como se usa?
1 - Se comienza cargando las librerías a utilizar
2 - Se carga una imagen a elección
3 - Se aplican distintos métodos para la disminución de niveles de profundidad. 
4 - Se utilizan los resultados para la creación de una nueva imagen
5 - Se visualizan las distintas imágenes creadas con distintos factores de cuantizacion para poder apreciar la técnica
6 - Se configuran y muestran los histogramas de la imagen original y de las imagenes donde se aplicó la cuantización.
